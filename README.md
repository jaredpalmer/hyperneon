# hyperneon

Fun theme for [Hyperterm](https://github.com/zeit/hyperterm).

## Installing

  1.  Download / Install [Hyperterm](https://github.com/zeit/hyperterm).
  2. Add `hyperneon` to `plugins` in ``~/.hyperterm.js`.
  3. Reload or restart Hyperterm
